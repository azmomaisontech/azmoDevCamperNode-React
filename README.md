# devCenterFullStack

Full stack application for a DEV Bootcamp Project.
To view, visit https://vast-brushlands-83538.herokuapp.com/

##Stack

```
React
CSS
Express(Node)
MongoDB
Heroku
```

## Usage

```
Rename "config/config.env.env" to "config/config.env" and update the values.
```

## Install Dependencies

```
npm install
```

## Run App

```
# Run in dev mode
npm run both
```
