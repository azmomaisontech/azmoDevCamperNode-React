# devCenterNodeJs

RestFUL API for a DEV Center Project.

##Usage

Rename "config/config.env.env" to "config/config.env" and update the values.

## Install Dependencies

```
npm install
```

## Run App

```
# Run in dev mode
npm run dev

# Run in prod mode
npm start
```

## Documentation

